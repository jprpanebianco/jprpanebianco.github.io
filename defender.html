Zombie Defender Prototype

Get the code

Idea

Zombie defender. Sans zombies. Substitute spider thingies and floating jellies. I mostly just wanted to see if using the mouse to aim was fun. I think it is, for the shotgun anyway. Also, watching spider thingies explode is gratifying, I am told.

Controls

Use the mouse to aim the character.
Use WASD keys to aim move the character.
Use the left mouse button to fire the selected weapon.
Use the right mouse button to toggle between machine gun, shotgun and laser.

Goal

Spiders damage the shelter if they touch it. Jellies fire bullets that damage the player. Keep the spiders away while avoiding the jelly bullets for as long as possible. The machine gun is good for killing the far away jellies. Shotgun kills close-up critters. Laser is over-powered. Switch weapons with right mouse button. Oh yeah, red spiders explode when they die, killing things around them, so kill them first.
